CESM1.2 Timing Table

General Notes:

  • This information is constantly subject to change due to changes in the model or machine hardware and software. Each entry contains information about the CESM version (tag name) and the date of the test.
  • The timing table results correspond to tag names that may not be officially released. The timing information is valid but the tag should not necessarily be used for scientific simulations. Please refer to the CESM website for a complete list of supported CESM versions.
  • This is a sortable table, click on the headings to perform a descending or ascending sort. (Note: the tool sorts based on character string, not numerical value).
  • A clickable link for each entry in the Compset column takes you to the raw timing output file.
  • Please contribute to this timing table by posting a raw CCSM timing file and information about your hardware (including a web link if possible) to the DiscussCESM Forums.
  • Please post questions regarding optimal PE layout for supported machines to the DiscussCESM Forums.

Component pe information is formatted as follows:

  • total_pes
  • tasks x threads
  • root_pe
