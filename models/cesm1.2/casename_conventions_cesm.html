CESM Experiment Case Name Convention

May 20, 2011

CSEG requests your cooperation in helping standardize CESM experiment case names. CSEG asks that you use the following standardized convention for your experiment case names as we move forwards. These standardized case names should be used for any experiments that

  • are added to the the CESM runs data base, or
  • have runs scripts in a shared CESM runs dir (at NCAR: /glade/proj3/cseg/runs), or
  • archive output in a shared CESM archive location (at NCAR: hpss:/CCSM)
In the future, the create_newcase script will have the ability to auto-generate case names conforming to this convention, but currently we ask that you create conforming case names manually when using the create_newcase script.

Thanks for your cooperation,
-CSEG

1. CESM1 Experiment Casenames

The convention is: <compset char>.<code base>.<compset shortname>.<res shortname>[.opt_desc_string].<nnn>[opt_char]

Examples:

      b.e10.BRCP26CN.f09_g16.001
      b.e10.BG1850CN.f19_g16.001
      b.e10.B1850CN.f19_g16.lig128ka.001
      b.e10.B1850.f19_g16.001
      b.e10.B1850.f19_g16.NRCM.001
      b.e10.B1850.f19_g16.NRCM.001a
      b.e10.B1850.f19_g16.NRCM.001b
      b.e10.B1850.f19_g16.NRCM.001c
      b.e10.B1850.f19_g16.DART.001
      c.e10.DTN460.f09_g16.001
      f.e10.B1850CN.f09_g16.001
      f.c40.FAMIPCN.f19_g16.001
      f.e10.FW.f45_g37.001
      f.e10.FGCN.f19_f19.001

Details:

  • <compset char> is a single character, the first letter of the compset, lower case
  • <code base> is "e10" for cesm1.0, "c40" for ccsm4.0, "c35" for ccsm3.5
  • <compset shortname> is the compset shortname recognized by create_newcase
  • is the resolution shortname recognized by create_newcase
  • [.opt_desc_string] is an OPTIONAL descriptive string, the only constraint is that it must not contain a "." (as this is reserved for delineating and parsing components of the case name). While this string is largely unconstrained, CSEG strongly encourages users to either create a very short string (eg. eight characters or less) or avoid using it altogether.
  • <nnn> is a 3 digit number, the smallest number that results in a unique case name
  • [opt_char] is an OPTIONAL single lower-case letter {a,b,...,z}, normally not used, but is allowed to denote a group of cases that are very closely related. Those cases would then have identical case names except for this single character suffix.

2. CESM1 Experiment Archival Location

CSEG also requests your cooperation in reserving the use of the archive location
    hpss:/CCSM/csm
for experiments that are likely to be of interest to the broader CESM community, and using other, working-group specific hpss directories...
    hpss:/CCSM/bgc
    hpss:/CCSM/paleo
    hpss:/CCSM/waccm
    hpss:/CCSM/<working group>
for experiments that are likely to be of interest to only one specific working group, and using a users home directory...
    hpss:/home/<user>
for experiments that are likely to be of interest to only one person.
