Known Problems

  

_____________________________________________________________________________________________

Description: Threading trouble with transient 20th Century mode for clm
Date       : 01/25/2010
Contact    : erik
Information: See bug 1102

CLM has problems using threading for transient land-use mode when it reads in
a new piece of dynamic land-use data (on year boundaries). Most examples of
this cause the model to die. It does seem to get past the problem if you run
CLM with OUT threading.
