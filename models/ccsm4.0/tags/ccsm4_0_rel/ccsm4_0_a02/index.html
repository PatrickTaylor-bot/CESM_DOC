Tag name: ccsm4_0_a02
Originator(s): cseg
Date: Wed Mar 10 2010
One-line Summary: Second ccsm4.0 alpha release tag.

This tag is the second released version of the ccsm4.0 alpha code.
The science is essentially the same, but because of numerous software 
improvements and bug fixes, users are strongly encouraged to upgrade 
from previous versions to this release.

- various minor bug fixes in both scientific and non-scientific code
- improvements to build and run scripts
- additional support for new machines
- improved default processor layouts for certain configurations
- improved parallel i/o library (pio)
- improved performance of high resolution configurations
- new support for omp threading in pop model
- clm Volatile Organic Compounds (VOC) module now implements science from
  MEGAN Heald et. al. 2008 version.
- new cam FV filter options needed for high res runs
- new cam dust tuning mods for trop_bam and trop_mozart configs
- new cam pre-industrial and 20th century transient SST datasets
- new cam atm_aero flag

Release tag ccsm4_0_a02 corresponds to internal release candidate ccsm4_0_rel05.
