Last Millennium (CCSM4)

CCSM4 CMIP5 Last Millennium forcing files (b40.lm850-1850.1deg.001)

The forcing files listed in Table 1 are linked to the SVN input data repository (https://svn-ccsm-inputdata.cgd.ucar.edu/).  You may register for a guest SVN account, or use an existing SVN account and password to download these files directly to your machine.

The Last Millennium control simulation was branched from year 301 of the CMIP5 1850 PI control simulation (b40.1850.track1.1deg.006).  The control simulation was run for 30 years, outputting high frequency daily output that was then used in an offline I-case simulation to spinup the land model carbon pools.  The high frequency output was turned off at year 30, and the control simulation was extended for 102 years.  The Last Millennium simulation branched from year 953 of the control simulation, using the carbon pools from the offline I-case simulation.  (Set up Notes)
