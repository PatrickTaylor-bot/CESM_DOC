List of changes

Robust changes

  • New topography for fininite volume (see C2)
  • Modifications to the dust (see C3d)
    New file for dust erodability + fine mode from 3.2% to 1.1% (Albani et al., 2014) + new optical properties
  • MG2 microphysics (see C4 and C4b)
  • New ice nucleation (see C5)
  • 32-level model (see C6)
  • MAM4 (see C7b)
  • New energy formulation (C8)
  • New remapping (C9)

Changes in testing

  • Gravity wave code (testing)
  • MAM mode width changes (to enable prognostic sulfur)
  • Prognostic stratospheric volcanic sulfate

Still in development

  • External forcings
    CH4 oxidation source for H2O
    OCS oxidation source for SO2
  • CCMI inertial gravity waves (IGWs)
    Currently the code related to the IGWs has a bug. It will be easily fixed, but will require re-tuning the model.
  • Convective gravity wave sources (tuning implications)
  • Consistent TMS: same parameter settings and orography files (?)
  • Convective transport and scavenging
