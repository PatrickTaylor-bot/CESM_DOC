CESM 1.2 Experiments, Data and Diagnostics

These simulations were completed using CESM 1.2. Refer to the Scientific Validations page to see a list of scientifically supported model configurations using CESM 1.2.

If you still have questions after reviewing the details of the model runs below, it is recommended that you contact the relevant CESM Working Group Liaison.

Important note regarding the CESM release used for the CESM CAM-chem simulations:

CESM1_2_2_CAMChem is a special tag that has been released for CAM-chem users. We have scientifically validated 5 compsets in this code based on F2000 compsets: FSSOA, FSTRATMAM3, FSTRATMAM4, and for specified dynamics compsets: FSDSSOA and FSDSMAM using MERRA reanalysis. All the compsets include full tropospheric and stratospheric chemistry for both CAM4-chem and CAM5-chem, and interactive biogenic emissions. The CAM4-chem compsets include secondary aerosol formation based on the 2-step scheme (FSSOA, FSDSSOA), while CAM5-chem uses a simple modal aerosol approach. Details and performed simulations are described here.

Compared to CESM1_2_2 we have added the capability and a compset to run with the modal aerosol model MAM4 and specified dynamics compsets (FSDSSOA, FSDSMAM (MAM3)).

Further, we have included bug fixes:
  1. SO2 Henry's law coefficient has been updated, which required a retuning of SO4, by increasing the in and below cloud solubility factor from 0.3 to 0.4
  2. MEGAN bug fixes, including using LAI averaged over the previous 10 days (instead of 1 timestep), and including the CO2 inhibition effect on isoprene emissions based on atmospheric values.
  3. Update on deposition velocities (Val Martin et al., 2014).
The CESM1_2_2_CAMChem release tag has limited support from CSEG on NCAR machine Yellowstone only. This tag has not been ported and tested on other platforms that CSEG supports.
