CESM1.2: CLM Documentation

Introduction

    The Community Land Model versions 4.0 and 4.5 in CESM1.2.0 are the latest in a series of land models developed through the CESM project. More information on the CLM project and access to previous CLM model versions and documentation can be found via the CLM Web Page.
